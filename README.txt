- Transporte
- Viagens
- Saúde
- Doação
- Loterias e Jogos
- Educação
- Água

Como rodar:
1. Extraia o ZIP.
2. Abra o arquivo index.html no navegador.
3. Os dados ficam salvos localmente via localStorage.

Observação:
Esta versão usa a chave localStorage gerenciador_despesas_v2_categorias.
Se você já usou a versão anterior, os dados antigos não serão misturados automaticamente.

2. Aguarde o GitHub Pages atualizar.
